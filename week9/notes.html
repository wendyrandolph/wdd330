Week 9 Notes

The Window Object

Global

A global variable is created without using "const, let, or var" and can be accessed from
anywhere in the program.

Dialogs

Three functions that produce dialogs in browsers: alert(), prompt(), confirm().

Which Browser?

There is a navigator property that returns a reference to the Navigator object.
ex: window.navigator.userAgent
<< "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_12_3) AppleWebKit/602.4.9 etc... however this
is not very reliable.

Location

window.location property is an object that contains info about the URL of the current page.
the href property returns the full URL as a string:
window.location.href
<< "https://www.sitepoint.com/premium/books/javascript ........ "
This property is a read/write property.
