Object Literals

In javascript it's a self contained set of related values and functions. If property value is a function then it's known as a method.

Think of it like a dictionary where you look up a property name and see a value.

  • They are flexible, they encapsulate code to be reused throughout a program.
  • Wrap an object in { }
  • Naming follows variables. ex. not "real name" not "real_name" yes "realName"
  • Properties & Methods can be changed or moved.

Keyboard Events

  • Keydown => occurs when pressed down and continues if key is held down.
  • Keypress => occurs after a keydown event.
  • Keyup => occurs when a key is released.

Keydown is an event action of pressing a key where, keypress event is the action of a character being typed on the screen.

Touchstart event is used for touchscreen devices.

  • touchend -> occurs when users stop touching.
  • touchmove -> already touched, moves around withouth leaving the screen.
  • touchenter -> already started touching surface.
  • touchleave -> still touching screen but leaves element.
  • touchcancel ->when touch event is interuppted.
