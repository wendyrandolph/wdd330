Week 2

Comments are highly important!

Well commented code allows any programmer especially yourself to understand whats going on.
There are two kinds of comments:

  1. //this is a short comment
  2. /* This is a longer comment ......... */

typeof = a special operator to find the type of value.

a few examples look like: typeof 'hello' <<'string' and typeof 10 << 'number'

Variables

Variables need to be declared before they can be used. You can see the value of a variable by typing it in the console.

Scope

Two kinds of scope:

  • Global scope --> accessible from everywhere in the program.
  • Local scope ---> defined within the block.


Variables Best Practices

  • camelCase
  • use sensible names
  • ex. _answer is better than x.
  • const instead of var

Types of Comparisons:

  • Equality
    • soft equality - (ex. answer == 5; << true)
    • hard equality - (ex. answer === 5; << true)
    • hard equality - (ex. answer === "5"; << false)
    • hard eqality - (ex. null === undefined; << false)
  • Inequality
    • soft inequality !=
    • hard inequality !==

Arrays

  • [An array is an ordered list of values]
  • Indicated by the use of []
  • You can add to, create, delete from arrays

Logic

  • If Statements

    • Syntax looks like:
    if (condition) {
    // code to run if the statement is true
    }

  • Else Statements

    • Syntax looks like:
    if (condition) {
    //code to run if statement is true
    } else {
    //code to run if statement is false
    }

  • Ternary Operator

    • Syntax looks like:
    condition ? (//code to run if true) : (// code to run if false)

  • Switch Statements

    • Basically you can string lots of if and else statements together.

Functions

  • Functions all have a name
  • Ninja programmers should always declare functions using function literals, function declarations, or function expressions
  • Invoking -( hello(); << 'hello world!' )
  • Keep your code DRY (Don't Repeat Yourself
  • Functions return values

    • function howdy() {
    return 'howdy world!';
